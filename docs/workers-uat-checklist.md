# 勞工模組功能驗收清單（UAT）

## A. 電話輸入區塊
- [ ] 可選國碼：`+86`、`+852`、`+853`
- [ ] 電話號碼僅允許數字
- [ ] 長度限制 7–11 位
- [ ] 錯誤時顯示紅色邊框與提示文案
- [ ] 行動版與桌面版排版正常（無重疊、無破版）

## B. 僱傭期限（月）
- [ ] 欄位標題為「僱傭期限（月）」
- [ ] 僅接受數字輸入
- [ ] 顯示「將儲存為：XX個月」
- [ ] 送出 payload 為 `XX個月`，不出現年/天等其他單位
- [ ] 狀態為「在職」時，會用赴港日期 + 月份計算到期日
- [ ] 到期前 9 個月會在業務概覽顯示續期提醒（姓名、餘下天數）

## B2. 勞工狀態規則
- [ ] 首次新增時狀態固定為「辦證中」
- [ ] 編輯切換到「在職」時，未填赴港日期不可保存
- [ ] 編輯切換到「離職」時，未填離港日期不可保存
- [ ] 編輯頁底部可查看「在港工作歷程」（批次資料）

## C. 文件上傳（存檔空間）
- [ ] 列表操作有「存檔」按鈕
- [ ] 打開後可切換：證件資料 / 學歷證明 / 工作證明
- [ ] 支援按鈕批次上傳
- [ ] 支援拖曳批次上傳
- [ ] 上傳中有進度條
- [ ] 類型限制：pdf/jpg/png
- [ ] 大小限制：單檔 <= 5MB
- [ ] 可刪除已上傳檔案

## D. 列表操作
- [ ] 列表每列提供：存檔 / 編輯 / 刪除
- [ ] 刪除需二次確認
- [ ] 刪除失敗會回滾列表狀態
