# 批文管理回歸測試清單

## A. 查詢與列表
- [ ] 可依批文編號搜尋
- [ ] 列表可正常顯示公司、部門、截止日期
- [ ] 配額數量顯示正確

## B. 新增 / 編輯
- [ ] 發證部門只能選固定選項
- [ ] 未選有效部門時阻擋提交並提示
- [ ] 發證日期不可大於今日
- [ ] 截止日期為唯讀且自動計算 `+12 個月`
- [ ] 可新增多個配額數量
- [ ] 配額序號同批文不可重複
- [ ] 刪除配額需二次確認

## C. 刪除 / 留痕
- [ ] 刪除批文正常
- [ ] 刪除配額有版本紀錄（quota_deleted）
- [ ] 切換部門有版本紀錄（department_changed）

## D. 業務概覽提醒
- [ ] 180±1 天提醒可產生
- [ ] 90±1 天提醒可產生
- [ ] 30±1 天提醒可產生
- [ ] 同區間不重複提醒
- [ ] 可標記已讀
- [ ] 可再次提醒

## E. 非影響驗證
- [ ] 匯出功能可正常使用（若現有）
- [ ] 列印功能可正常使用（若現有）
