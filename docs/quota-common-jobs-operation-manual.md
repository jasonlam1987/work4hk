# 申請配額 - 常見職位新申請操作手冊

## 1. 模組入口
- 進入 `申請配額`。
- 開啟一筆申請後切換到 `常見職位新申請` 分頁。

## 2. 操作流程
1. 先在 `申請常見職位` 分頁新增職位（職位1...職位N）。
2. 回到 `常見職位新申請`，在 `職位選擇` 輸入關鍵字搜尋並選取職位。
3. 選擇 `輪班需求`，填寫每日工作時間區間。
4. 先完成 `會話`（粵語/英語/其他語言）等級，再完成 `讀寫` 等級；等級為 `無需/略懂/一般`。
5. 若 `其他語言` 選擇 `略懂/一般`，需輸入語言名稱。
6. 在 `技能與其他要求` 區輸入內容，可使用 `Bold` 與 `Bullet`。
7. 點 `儲存草稿`（僅驗證必填）或 `提交申請`（全欄位驗證）。

## 4. QA 建議測試清單
- 切換職位 `職位1 -> 職位N` 後，工時/語文/技能欄位應重置。
- 輸入 `23:00 -> 02:00` 顯示跨日提示且可提交。
- 會話或讀寫任一語言未選等級應阻擋提交；其他語言為略懂/一般且未填名稱應阻擋提交。
- 技能欄位超過 500 字應即時報錯且不可提交。
- 鍵盤操作：Tab 可遍歷職位選擇、輪班、時段、語言等級單選、RTF 工具按鈕與提交按鈕。
