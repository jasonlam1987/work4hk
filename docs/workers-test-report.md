## 3. 錯誤情境覆蓋
- 必填欄位缺值：前端提交前阻擋（中文名、身份證、僱主、批文）。
- 電話格式錯誤：前端即時驗證（7–11 位數字）。
- 檔案類型不符：前端阻擋（僅 pdf/jpg/png）。
- 檔案大小超標：前端阻擋（單檔 <= 5MB）。
- 資料庫連線中斷：前端會顯示後端錯誤訊息並保留表單資料。

## 4. 注意事項
- 後端資料庫中斷屬整合測試場景，需在後端環境（非本前端倉庫）透過故障注入/關閉 DB 連線進一步驗證。
